Асінхронне перемикання розкладок клавіатури

Перемикання:

  • CapsLock - English-US
  • Ctrl+CapsLock - Українська
  • Shift+CapsLock - русский
  • Alt+CapsLock - English-US (Dvorak)
  • Ctrl+Shift+CapsLock - CapsLock

Чому?

Бо зручно. Немає циклічного перемикання між розкладками: комбінація дорівнює певній розкладці.

Можливо комусь це здасться незручним, але це не так. Особистий досвід використання цієї конфігурації десь з 2005-го року це доводить :)
